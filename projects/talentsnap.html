Plugin UI

TalenSnap has a solid UI, but we found some ways to improve it.

1. Separating the nav into primary tasks like ‘FitScore’ and secondary tasks like ‘Settings’ makes it easier to focus on what’s important.

2. Re-naming ‘Filters’ and removing the icons makes it instantly understandable.

3. Adding more information to critical form fields so users understand what’s required without having to guess.
